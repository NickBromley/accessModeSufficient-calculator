EPUB accessModeSufficient Metadata Calculator

A simple tool to calculate accessModeSufficient metadata values for an epub, depending on the type of content it contains.

Select the types of content in your epub using the checkboxes below. Then select what text/media alternatives, if any, are included.

The required combinations of accessModeSufficient values you need to include in the epub package metadata will be listed underneath.
